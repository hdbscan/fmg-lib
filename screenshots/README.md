# Screenshot Artifacts

- `milestones/`: point-in-time UI captures plus JSON metadata.
- `drift/`: side-by-side comparison artifacts for local vs upstream or local vs approved baseline review.
- `ui-dummy-step-01.png`: approved initial dummy baseline captured before the generic workflow was added.

Use `bun run screenshot:milestone` for new milestones and `bun run screenshot:drift` for review comparisons.
